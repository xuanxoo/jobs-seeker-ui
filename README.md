# Search Form

Frontend challenge implemented with hexagonal architecture with React and Typescript.

## Requisites

`node >= 14`

# Installation as default app

`$ yarn install`

## Development

`$ yarn start`

Open [http://localhost:3005/](http://localhost:3005/) to view it in the browser.

## Tests

#### Unit tests

`$ yarn test`
